Module Naive_bayes

module Naive_bayes: sig .. end

Naive bayes algorithm

val predict : Matrix.t -> Stdlib.Float.t list -> Stdlib.Float.t list

predict x y fits the naive bayes model using features x and target y then returns the predictions for the features x

val fit_and_predict : Matrix.t ->
       Matrix.t -> Stdlib.Float.t list -> float list -> float * float * float list

fit_and_predict x_train x_test y_train y_test fits the naive bayes model using features x_train and target y_train then predicts the targets of x_test, and returns the (accuracy, mean-squared error, y_pred) when comparing this prediction to the actual y_test
